# 📊 YouTube Data Dashboard
YouTube Data Dashboard using Streamlit and YouTube API

## 🚀 Project Overview

The **YouTube Data Dashboard** is an interactive web application built using **Streamlit** that allows users to analyze the performance of any YouTube channel. By leveraging the **YouTube Data API v3**, this dashboard provides real-time insights into video metrics such as views, likes, and comments.

---

## 🎯 Features

* 📊 Real-time YouTube data fetching
* 📈 Interactive visualizations (Bar & Line Charts)
* 📋 Tabular data display of videos
* 🏆 Top performing videos analysis
* 🎯 Filter videos based on minimum views
* 📌 KPI Metrics (Total Views, Likes, Comments)
* 🔐 Secure API key management using `.env`
* ⚠️ Error handling for invalid inputs

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/youtube-data-dashboard.git
cd youtube-data-dashboard
```

---

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

### 3️⃣ Add API Key

Create a `.env` file in the root directory:

```
API_KEY=your_youtube_api_key_here
```

---

### 4️⃣ Run the Application

```
streamlit run app.py
```

---

## 📌 Usage

1. Enter a valid **YouTube Channel ID** in the sidebar
2. View:

   * Total Views, Likes, Comments
   * Video performance table
   * Top performing videos
   * Filtered results
   * Visual charts
